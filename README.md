# Enterprise Task Manager

Sistema de gestión de tareas diseñado para optimizar el flujo de trabajo de equipos ágiles. Este proyecto simula un entorno de producción real, enfocado en la escalabilidad, mantenibilidad y código limpio sin dependencia excesiva de frameworks.

## Stack Tecnológico

* **Backend:** PHP 8+ (Nativo)
* **Frontend:** HTML5, CSS3, Javascript (Vanilla)
* **Base de Datos:** MySQL / MariaDB
* **Control de Versione:** Git y GitHub (Git Flow)

## Funcionalidades Clave (En desarrollo)

* [ X ] Autenticación de usuarios (Login/Registro) seguro.
* [ X ] Dashboard de administración de tareas (CRUD).
* [ X ] Asignación de roles y permisos.
* [ X ] Interfaz reactiva sin frameworks pesados.

## Estructura del Proyecto

El proyecto sigue una arquitectura modular organizada de la siguiente manera:

/assets     -> Recursos estáticos (CSS, JS, Imágenes)
/includes   -> Lógica de backend reutilizable y conexión a la BD
index.php   -> Punto de entrada de la aplicación
